Detailed Description

DHState objects are used to generate key pairs and perform Diffie-Hellman calculations.

Performs a Diffie-Hellman calculation.

Parameters
statePoints to the DHState.
shared_keyPoints to the shared key on exit.
shared_key_lenThe length of the shared_key buffer in bytes.
private_keyPoints to the private key.
private_key_lenThe length of the private_key in bytes.
public_keyPoints to the public key.
public_key_lenThe length of the public_key in bytes.
Returns
NOISE_ERROR_NONE on success, NOISE_ERROR_INVALID_PARAM if state, shared_key, private_key or public_key is NULL, or NOISE_ERROR_INVALID_DH_KEY if either public_key or private_key are invalid for the algorithm.

If the input public_key is the special null value, then the output shared_key will also be the null value.

Generates a key pair for a Diffie-Hellman algorithm.

Parameters
stateThe DHState object.
private_keyPoints to the private key on exit.
private_key_lenThe length of the private_key buffer in bytes.
public_keyPoints to the public key on exit.
public_key_lenThe length of the public_key buffer in bytes.
Returns
NOISE_ERROR_NONE on success, NOISE_ERROR_INVALID_PARAM if state, private_key or public_key is NULL, or NOISE_ERROR_INVALID_LENGTH if private_key_len or public_key_len is incorrect.
Note
This function needs to generate random key material for the private_key, so the system random number generator must be properly seeded before calling this function.
