void* noise_new_object ( size_t  size)

Allocates memory from the system for an object.

Parameters
sizeThe number of bytes of memory to allocate for the object.
Returns
Pointer to the allocated memory or NULL if the system is out of memory.

If size is greater than or equal to sizeof(size_t), then the first few bytes in the returned memory will be set to size. That is, the object is assumed to start with a size field. All other bytes in the object are initialized to zero.

Note
If the caller is allocating a structure, then noise_new() is a better option to ensure type-safety.
See Also
noise_new(), noise_free()
